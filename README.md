# AutoDefineSymbolsBasedOnPackages
Generate Define Symbols based on packages installed in the project

You can manually detect all symbols from `Tools > AutoSymbolDefiner > Detect All Defines`

### How to Import:
 - Add to manifest.json
```
"com.arikan.autosymboldefiner" : "https://github.com/bilal-arikan/AutoDefineSymbolsBasedOnPackages.git",
```
